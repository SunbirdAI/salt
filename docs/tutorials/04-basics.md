### one-to-multiple translation: English text to Luganda and Acholi text

```python

import sys
sys.path.append('../..')
import leb.dataset
import leb.utils
import yaml

```

set up the configs

```python

yaml_config = '''
huggingface_load:
  path: Sunbird/salt
  split: train
  name: text-all
source:
  type: text
  language: eng
  preprocessing:
      - prefix_target_language
target:
  type: text
  language: [lug, ach]
'''

config = yaml.safe_load(yaml_config)
ds = leb.dataset.create(config)
list(ds.take(5))

```
